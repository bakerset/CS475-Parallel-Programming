# Ecosystem Simulation Data Export Variant

This folder contains an expanded variant of the OpenMP ecosystem simulation focused on structured output and post-processing. It keeps the same core model as project 2 while adding formatted simulation output and CSV conversion support for portfolio charts or reports.

## Highlights

- Extends the ecosystem simulation with file-based reporting.
- Produces structured simulation data suitable for visualization.
- Includes a Python conversion helper for CSV-oriented workflows.

## Key Files

- `baker-proj2.cpp` - Ecosystem simulation source with output support.
- `convertcsv.py` - Helper script for converting simulation output to CSV.
- `makefile` - Builds the project executable.

## Build

```bash
make
```
